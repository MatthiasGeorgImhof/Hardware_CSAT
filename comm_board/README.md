# Communication Board #


Communication board with ethernet and radio operating in the US ISM band of 915MHz. Check your jurisdiction about availability of this band!

- STM32L496 microprocessor
- TI CC1101 transceiver for 915MHz oeration and TI CC1190 power frontend
- Ethernet modem
